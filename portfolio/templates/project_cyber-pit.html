CyberPit is my first larger project in Python - my playground from early days. I used to play with classes, data structures, app design concepts. I spent most of the time refactoring. I learned simple user management, SQL DB interaction with raw SQL statements, terminal interaction and much more. Game is fully playable, although it might need some fine-tuning and user testing. Maybe one day...

You play as a techie owning a battle robot, meeting other robots in the arena to fight. Each robot is unique and has unique set of weapons, dealing different points of damage. You get to buy a robot at the start of the game. For each victory you gain more credits and can afford better bot. Bot selection is limited, but can be easily enriched by manually adding custom bots to json file.

Have fun playing!

Technologies used

  • Python
  • SQLite
